# Top 100 Claude Code Skills & Skill Packs — April 2026

A curated showcase of the most popular new Claude Code skills and skill packs trending on GitHub and in the plugin marketplace.

**Live site:** https://alimabsoute.github.io/claude-skills-100/

## What's inside

- **100 hand-curated entries** — a mix of skill packs (1000+ skill mega-libraries) and individual specialized skills.
- **20 categories** — from quantum computing and CFD to full-stack engineering and agent orchestration.
- **Full detail cards** — each with a tagline, description, extended explanation, and three concrete use cases.
- **Direct GitHub links** to every project.

## How entries were chosen

1. **GitHub research** — searched for `claude-code-skills`, `SKILL.md`, `topic:agent-skills`, and cross-referenced awesome-lists.
2. **Popularity signals** — stars, recency of commits, citations in aggregator lists.
3. **Distinctiveness** — filtered out skills that are near-duplicates of already-popular generic skills.
4. **Coverage** — intentionally included niche domains (quantum, bioinformatics, robotics, embedded, CFD, Salesforce, etc.) that most curated lists skip.

## Updating

Entries live in `data.js` as a single `window.SKILLS` array. Edit that file, commit, push — GitHub Pages picks up changes in about a minute.
